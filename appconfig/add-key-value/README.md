## Add a Key/Value to Azure App Configuration

To add a Key/Value to the Azure App Configuration use the command line below (`yes` does not prompt for confirmation):

```shell
  az appconfig kv set \
    --name $APP_CONFIG_NAME \
    --key myKey \
    --value "My value" \
    --yes
```
