## Create the namespace

To create the namespace use the following command line:

````shell
  export EVENTHUBS_NAMESPACE=eventhubs-$RANDOM

  az eventhubs namespace create \
     --name $EVENTHUBS_NAMESPACE \
     --resource-group $RESOURCE_GROUP \
     --location $REGION
````
