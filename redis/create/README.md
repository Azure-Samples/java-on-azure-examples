## Create the Azure Cache for Redis

To create the Azure Cache for Redis use the following command line:

````shell
  export REDIS_NAME=joaz-redis-$RANDOM

  az redis create \
    --location $REGION \
    --name $REDIS_NAME \
    --resource-group $RESOURCE_GROUP \
    --sku Basic \
    --vm-size c0
````
